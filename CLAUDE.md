# PM Job Fetcher — Claude Code Instructions

You are helping a user set up and run their PM Job Fetcher. They may not be technical.

## What this project does

Checks 44+ company career pages for Product Manager jobs, saves results to a markdown file, and optionally sends a Slack notification.

## How to help users add companies

When the user wants to add companies (by name, URL, or pasting a list/spreadsheet):

1. Parse their input into company names
2. Run `python3 add_companies.py CompanyA CompanyB CompanyC` to auto-detect each company's ATS
3. If a company isn't found, ask the user for the company's careers page URL (e.g., boards.greenhouse.io/stripe)
4. Run `python3 add_companies.py "https://the-url-they-gave"`

To see current companies: `python3 add_companies.py --list`
To start fresh: `python3 add_companies.py --clear`

## How to help users set up Slack notifications

1. Tell them to go to: https://api.slack.com/messaging/webhooks
2. Walk them through: Create App > From Scratch > name it "Job Alerts" > pick workspace
3. Go to "Incoming Webhooks" > toggle ON > "Add New Webhook to Workspace" > pick channel
4. Copy the webhook URL
5. Edit settings.json and put the URL in "slack_webhook_url"

## How to run

- Fetch new jobs: `python3 fetch_jobs.py`
- Show all jobs (ignore history): `python3 fetch_jobs.py --all`
- Reset history: `python3 fetch_jobs.py --reset`

## Key files

- `companies.json` — list of companies to check (auto-generated by add_companies.py)
- `settings.json` — optional Slack webhook URL
- `seen_jobs.json` — tracks which jobs have been shown already
- `output/` — markdown files with job listings

## Important

- This uses ONLY Python standard library. No pip install needed.
- The user just needs Python 3.6+ installed.
- Be patient and explicit — users may not know terminal basics.
